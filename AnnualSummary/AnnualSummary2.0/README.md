# AnnualSummary2.0

## 一、目的

* 前端年度总结

## 二、技术

* [Vue.js](https://cn.vuejs.org/)

* [Node.js](https://nodejs.org/en/)

## 三、开发工具

* [VsCode](https://code.visualstudio.com/)

## 四、 功能

* 大前端成员展示

* 全年项目总数

* 个人所占各个事业部的项目总数

* 参与人数最多的项目

* Labels各个类型对应项目的数量(空挡、正常、空闲、紧急、实验室、任务重)

* 多雷达图展示各个事业部的项目数

* Pause的项目

* 设置了结束日期的项目
